Features

  • Moderation features

    • Rename attention seekers
    • Purge messages
  • Fun commands
    • Shiba Inu pictures
    • Dog facts
    • Anime search
  • Utilities
    • Vanity role creation
    • Calculator
    • Show avatar
    • Server/Discord join date

  • Tagging features
    • Tag creation
    • Tag information (usage count)

    • Jisho (Japanese dictionary)
    • UrbanDictionary
    • Jumbo custom emoji
  • More moderation features!
